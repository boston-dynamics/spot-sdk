Bases: bosdyn.client.exceptions.Error

exception bosdyn.client.exceptions.InvalidRequestError(response, error_message=None)[source]¶

Bases: bosdyn.client.exceptions.ResponseError

The provided request arguments are ill-formed or invalid.

This is programmer error, hence it should be fixed and not ignored. This error does not depend on the state of the system.
